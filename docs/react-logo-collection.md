## Getting Started

Install React Logo Collection to your project:

`yarn add @react-pakistan/react-logo-collection` or `npm install @react-pakistan/react-logo-collection`

- ### LogoBitcoinColor29
```javascript
  import { LogoBitcoinColor29 } from ‘@react-pakistan/react-logo-collection/logos/bitcoin-color’;

  <LogoBitcoinColor29
    fontSize="32px"
    color="#2E2E2E"
  />
```
